Fórmulas de Propagación de Errores
Suma y Resta:

Error = √(ex² + ey²)

Producto y División:

Error relativo = √((ex/x)² + (ey/y)²)

Casos especiales:
  • Para división: Y ≠ 0
  • Para producto: X ≠ 0 y Y ≠ 0
  • Error absoluto = |Error relativo| × |Resultado|
